**11. Angry & Firm Replies**
1. Leave me alone. / मला एकटे सोडा. / मुझे अकेला छोड़ दो।
2. Not in the mood. / मूडमध्ये नाही. / मूड नहीं है।
3. Don't test my patience. / माझी संयमाची परीक्षा घेऊ नका. / मेरी सहनशीलता मत परखो।
4. Stop it right now. / आत्ताच थांबवा. / अभी बंद करो।
**12. Smart & Quick Chat Replies**
1. Noted! / नोंद घेतली! / समझ गई!
2. Sounds like a plan. / एक योजनेसारखे वाटते. / बढ़िया आइडिया है।
3. Let's do it. / चला ते करूया. / चलो, करते हैं।
4. Makes sense. / अर्थपूर्ण. / बात सही है。
5. Wait a sec. / एक सेकंद थांब. / ज़रा रुको。
