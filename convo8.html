**8. Apology & Forgiveness**
1. Past is past, let's move on. / भूतकाळ गेला, चला पुढे जाऊया. / जो बीत गया, उसे जाने दें।
2. I forgive you, no grudges. / मी तुला माफ करते, कोणतेही द्वेष नाही. / मैं तुम्हें माफ़ करती हूँ, कोई शिकायत नहीं।
3. I'm really sorry. / मला खरोखर माफ करा. / मुझे सच में माफ़ कर दो।
4. Please forgive me. / कृपया मला माफ करा. / कृपया मुझे माफ़ कर दो।
**9. Requests & Favors**
1. Please listen to me. / कृपया माझे ऐका. / कृपया मेरी बात सुनो।
2. Can you do this for me? / तू हे माझ्यासाठी करू शकतोस का? / क्या तुम मेरे लिए ये कर सकते हो?
3. I really need this. / मला याची खरोखर गरज आहे. / मुझे सच में इसकी ज़रूरत है।
**10. Playful Replies**
1. Oh really? / अरे खरंच? / अच्छा जी।
2. Liar! / खोटा! / चल झूठे।
3. I knew it. / मला माहित होतं. / मुझे पता था।
4. You're trouble. / तू त्रास आहेस. / तुम तो मुसीबत हो।
